# 🌦️ Weather App (Python)

This is a simple Python-based weather application that fetches real-time weather data using an API.

## 🔧 Features

* Fetches live weather data
* Shows temperature, humidity, and weather conditions
* User input-based city search
* Error handling for invalid cities

## 🛠️ Tech Used

* Python
* Requests library
* OpenWeather API

## ▶️ How to Run

1. Install requests:
   pip install requests

2. Run the program:
   python weather.py

## 📌 Note

You need an API key from OpenWeather to run this program.
